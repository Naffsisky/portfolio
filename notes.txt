Kubernetes K3S Med-Small Deployment

CONTROL PLANE

Nonaktifkan Memory SWAP

sudo swapoff -a
sudo sed -i '/ swap / s/^/#/' /etc/fstab

sudo apt-get update
sudo apt-get install -y containerd
sudo systemctl status containerd

Install K3s
curl -sfL https://get.k3s.io | sh -

sudo k3s kubectl get node

Ambil Token
sudo cat /var/lib/rancher/k3s/server/node-token

Buat Namespace
sudo kubectl create namespace frontend

sudo kubectl get namespaces

Buat deployment.yaml (Pastikan image sudah ada di registry)
sudo vim frontend-deployment.yaml

apiVersion: apps/v1
kind: Deployment
metadata:
  name: react-aksa
  namespace: frontend
spec:
  replicas: 2
  selector:
    matchLabels:
      app: react-aksa
  template:
    metadata:
      labels:
        app: react-aksa
    spec:
      nodeSelector:
        kubernetes.io/hostname: frontend   # pastikan node "frontend" ada dan benar
      containers:
      - name: react-aksa-container
        image: napzky/test-nap:latest      # ganti sesuai image di Docker Hub kamu
        ports:
        - containerPort: 80                # NGINX atau web server default port 80

Buat service.yaml (Pastikan image sudah ada di registry)
sudo vim frontend-service.yaml

apiVersion: v1
kind: Service
metadata:
  name: react-aksa-service
  namespace: frontend
spec:
  selector:
    app: react-aksa
  ports:
    - protocol: TCP
      port: 80
      targetPort: 80
  type: NodePort

Pastikan Hostname sudah benar dan connect

AKTIFKAN DI SERVER WORKER
curl -sfL https://get.k3s.io | K3S_URL=https://[IP]:6443 K3S_TOKEN=[TOKEN] sh -

curl -sfL https://get.k3s.io | K3S_URL=https://34.101.51.213:6443 K3S_TOKEN=K106767ba7b1a350ab06bc65f82a55e64d882024e15eecbf3d2b99d6e97d7075621::server:fb5ff4129165eb70152847a95c593075 sh -


DEPLOY DI CONTROL PLANE

sudo kubectl apply -f frontend-deployment.yaml

sudo kubectl apply -f frontend-service.yaml

sudo kubectl get pods -n frontend

sudo kubectl get svc -n frontend

kubectl get pods -n frontend -o wide

sudo kubectl get hpa -n frontend

kubectl apply -f https://github.com/kubernetes-sigs/metrics-server/releases/latest/download/components.yaml
